# 📱 SMS Helper – Android App

**SMS Helper** is an Android application that allows users to locate their misplaced phones using simple SMS commands — even when the phone is in **silent mode** or has **no internet connection**.

---

## 🚀 Features

- 🔍 Locate lost phones remotely via SMS
- 🔕 Automatically switches from **silent** to **ringing** mode
- 🔊 Plays a loud ringtone to help users find their device
- ✅ Works **offline** (no need for internet or data)
- 🔒 Secure: only responds to predefined SMS command
- 🔧 Lightweight and battery-friendly

---

## 💡 How It Works

1. The app listens for incoming SMS using a **BroadcastReceiver**.
2. When it receives a specific command (e.g., `FIND_PHONE`), it:
   - Switches the phone to **normal (ringing)** mode
   - **Plays a ringtone** at full volume
3. This helps the user locate the phone even if it’s lost, silent, or offline.

---

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** Android Studio
- **Backend (Optional):** Firebase (for storing logs or user data)
- **Core Android Concepts:** BroadcastReceiver, AudioManager, MediaPlayer, Runtime Permissions

---

## 🔐 Permissions Used

- `RECEIVE_SMS`
- `READ_SMS`
- `MODIFY_AUDIO_SETTINGS`
